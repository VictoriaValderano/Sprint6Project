# Descrição do Projeto
Este projeto realiza uma análise completa de dados históricos de vendas de videogames para a loja online Ice. O objetivo é identificar padrões que determinam o sucesso de um jogo e planejar campanhas publicitárias estratégicas para 2017.
A análise utiliza dados de vendas, avaliações de usuários e críticos, informações sobre plataformas, gêneros e classificações ESRB para fornecer insights acionáveis sobre o mercado de videogames.

# Objetivos

Identificar padrões que determinam se um jogo tem sucesso ou não
Analisar o ciclo de vida de plataformas de videogames
Determinar os gêneros mais lucrativos
Criar perfis de usuários por região (América do Norte, Europa e Japão)
Testar hipóteses sobre avaliações e vendas
Fornecer recomendações para campanhas publicitárias em 2017

# Dataset
O dataset contém informações sobre vendas de videogames de diferentes plataformas e regiões:

## Colunas:

Name: Nome do jogo
Platform: Plataforma (ex: Xbox, PlayStation)
Year_of_Release: Ano de lançamento
Genre: Gênero do jogo
NA_sales: Vendas na América do Norte (milhões USD)
EU_sales: Vendas na Europa (milhões USD)
JP_sales: Vendas no Japão (milhões USD)
Other_sales: Vendas em outros países (milhões USD)
Critic_Score: Pontuação de críticos (máximo 100)
User_Score: Pontuação de usuários (máximo 10)
Rating: Classificação ESRB

## Características:

16.715 registros
Dados de 1980 a 2016 de múltiplas plataformas e regiões.
